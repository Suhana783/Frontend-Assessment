# Frontend Assessment

A simple carousel slider project built with HTML, CSS, and JavaScript.

## Features

- Interactive slide carousel with navigation buttons
- Multiple slides (Home, Shoes, Fashion, Clothes)
- Left and right arrow navigation
- Clean and responsive design

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and layout
- `script.js` - Carousel functionality

## Usage

Open `index.html` in your browser and use the arrow buttons to navigate between slides.
